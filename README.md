# jsonbin
This repository is meant as a persistent JSON store. The ultimate goal is to provide both public and private-only access options. One way to do it is to keep this repo private while serving the JSON-blob over published GitHub pages! That way, one cannot access a given JSON blob without either its pathname (address) or read-only access to this repo. 

However, currently, only paid GitHub plans offer publishing GitHub pages from private repos. For now, therefore, this repo is kept public so that GitHub pages can function.
